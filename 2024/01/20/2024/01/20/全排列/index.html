leetcode46：给定一个不含重复数字的数组 nums ，返回其所有可能的全排列 。你可以 按任意顺序 返回答案。

看到这道题第一个想法是数据结构学的使用辅助数组visited来递归搜索，当visited都为1（全被访问）就是一个全排列

class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        def backtrack(res, nums, visited):
            if len(res) == len(nums):
                result.append(res.copy())
                return

            for i in range(len(nums)):
                if visited[i]:
                    continue
                res.append(nums[i])
                visited[i] = 1
                backtrack(res, nums, visited)
                res.pop()
                visited[i] = 0

        result = []
        visited = [0] * len(nums)
        backtrack([], nums, visited)
        return result

就地搜索，bactrack传入两个参数,一个是起始位置，一个是列表的长度。
在每一个调用的bactrack中，循环只做一件事，那就是改变start位置的元素（通过交换实现）
而递归出口便是start到达最后一个元素，即start=end时，形成一个全排列。
如此，简单明了，一目了然。

class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        def backtrack(start, end):
            if start == end - 1:
                result.append(nums[:])

            for i in range(start, end):

                nums[start], nums[i] = nums[i], nums[start]
                backtrack(start+1, end)
                nums[i], nums[start] = nums[start], nums[i]

        result = []
        backtrack(0, len(nums))
        return result

leetcode47：给定一个可包含重复数字的序列 nums ，按任意顺序 返回所有不重复的全排列。

就地搜索，bactrack传入两个参数,一个是起始位置，一个是列表的长度。
在每次调用bactrack中，循环只做一件事，那就是改变start位置的元素（通过交换实现），交换过的元素会被记录到集合seen中，如果发现后续的元素已经在seen中出现，那么跳过本次循环不发生元素交换。
而递归出口便是start到达最后一个元素，即start=end时，形成一个全排列。
如此，简单明了，一目了然。

class Solution:
    def permuteUnique(self, nums: List[int]) -> List[List[int]]:
        def backtrack(start, end):
            if start == end - 1:
                result.append(nums[:])

            seen = set()
            for i in range(start, end):
                if nums[i] in seen:
                    continue

                seen.add(nums[i])
                nums[start], nums[i] = nums[i], nums[start]
                backtrack(start+1, end)
                nums[i], nums[start] = nums[start], nums[i]



        result = []
        backtrack(0, len(nums))
        return result

总结全排列两种情况

对于无重复元素的数列来说，每个位置上元素都可能出现一次，但是对于重复元素的数列来说，每个位置只能出现一次相同的元素，这样就可以保证结果不会出现重复的排列。

组合

给定两个整数 n 和 k，返回范围 [1, n] 中所有可能的 k 个数的组合。

在全排列的基础上实现组合数，就是不再考虑以及选择过的元素，通过 idx标定。

class Solution:
    def combine(self, n: int, k: int) -> List[List[int]]:
        
        def backtrack(idx):
            if len(res) == k:
                result.append(res[:])
                return
            for i in range(idx, n+1):
                res.append(i)
                backtrack(i+1)
                res.pop()
       
        result = []
        res = []
        backtrack(1)
        return result
